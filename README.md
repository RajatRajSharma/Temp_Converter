## How to Use

1. Open the [Temperature Converter](https://rajat-temp-converter.onrender.com/) website.
2. Enter the temperature value in the input field.
3. Select the source temperature unit (째F or K) from the dropdown.
4. Click the "Convert" button to see the equivalent temperature in Celsius (째C).
5. The result will be displayed below the input form.
